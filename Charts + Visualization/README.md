# Day 12 Charts + Visualization

This folder contains the Day 12 charting and visualization experience for the Developer Portfolio Evaluator.

## Features

- Radar chart for five portfolio score dimensions
- Bar chart for language usage distribution
- Heatmap-style activity overview by weekday
- Built with React and Chart.js using Vite

## Run locally

Open a terminal in the `day12` folder and run:

```bash
npm install
npm run dev
```

Then open the Vite URL shown in the terminal.

## Build

```bash
npm run build --workspace client
```
