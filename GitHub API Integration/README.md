# Day 4 - GitHub API Integration

## Setup
1. npm install
2. Update .env with your MongoDB URI and GitHub token
3. npm run start

## Endpoints
- GET /api/health - Health check
- GET /api/github/user/:username - Get user profile
- GET /api/github/repos/:username - Get user repos
- POST /api/reports - Create a report
- GET /api/reports - Get all reports

## Test in Postman
Use the above endpoints to test GitHub API integration and MongoDB operations.
