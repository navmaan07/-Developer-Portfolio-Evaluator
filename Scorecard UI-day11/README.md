# Day 11 Scorecard UI

This folder contains the Day 11 scorecard UI for the Developer Portfolio Evaluator.

## Features

- Profile detail card with avatar, role, and metadata
- Circular overall score indicator
- Metric cards for activity, code quality, and community
- Responsive styling for mobile and desktop screens

## Run locally

Open a terminal in the `day11/client` folder and run:

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Build

```bash
npm run build
```
